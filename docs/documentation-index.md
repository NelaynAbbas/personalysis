## Glossary

A quick reference for key terms used throughout PersonalysisPro:

- **Survey**: A collection of questions designed to gather specific information
- **Response**: A completed submission to a survey
- **Trait**: A personality characteristic measured through survey responses
- **Dashboard**: A visual display of key metrics and insights
- **Segment**: A defined subset of respondents based on specific criteria
- **Integration**: A connection between PersonalysisPro and external services
- **API Key**: A unique authentication credential for API access
- **Webhook**: An HTTP callback that delivers real-time notifications
